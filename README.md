# 3D PacMan

# A modern 3D reimagining of the classic Pac-Man arcade game, built in Unity. Navigate a fully rendered maze, collect pellets, avoid ghosts, and survive as long as possible



# How to Play

# W / A / S / D . Move your character around the maze

# Mouse . Control the camera view

# Goal . Collect all pellets while avoiding enemies

# (Not implemented yet) Power-Ups . Some pellets may grant temporary advantages.




# Features

# Fully 3D environment

# Classic Pac-Man gameplay with a modern twist

# Smooth character movement and camera controls

# Enemy AI that "chases" the player




# Built With

# Unity (version number here)

# C# scripts for gameplay logic

# 3D models and materials created/used for the environment and characters



# Installation

# Clone or download the repository

# Open the project in Unity

# Press Play to run the game in the Unity Editor

# (Optional) Build the game for your platform through File > Build Settings
